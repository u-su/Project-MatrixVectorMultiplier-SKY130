# AXI-Steam Matrix Vector Multiplier with UART Interface on SKY130

A Matrix-Vector Multiplier digital circuit with AXI-stream flow control and an UART serial interface was developed from scratch on RTL level using SystemVerilog HDL by integrating elementary logic circuits such as Counter, Accumulator, Parallel multipliers, Skid buffer, Parallel-to-serial & Serial-to-parallel bus converters, AXI-S control elements and a simple UART transceiver. 

Design was simulated, synthesized and validated on Xilinx Zynq-7000 FPGA (Digilent Zybo) using Xilinx Vivado ML Edition.
